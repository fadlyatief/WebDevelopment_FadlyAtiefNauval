# WebDevelopment_FadlyAtiefNauval
DummyForms merupakan web-based form builder yang kedepannya memungkinkan pengguna membuat formulir kustom, membagikannya secara online, dan mengumpulkan respons dalam satu sistem terpusat. Proyek ini dikembangkan sebagai latihan pengembangan web dengan fokus pada fitur, UI, dan alur pengguna.
